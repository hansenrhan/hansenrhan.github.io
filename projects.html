Zillow Scraping
Finance

The zillow_scraper Python package was created to automate the process of gathering real estate listings from Zillow. Designed to work with both rental and for-sale properties across multiple locations, the tool collects detailed attributes for each listing. This comprehensive dataset serves as a valuable resource for users interested in market trend analysis, investment evaluation, and housing policy research.

Tools: Python (bs4, selenium)
